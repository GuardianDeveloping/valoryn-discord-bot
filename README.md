# ⚔️ Valoryn

Valoryn is a fantasy RPG Discord bot that transforms your server into a living guild hall where members can level up, earn renown, collect loot, conquer dungeons, and forge their legend.

## Features

### 🏆 Progression

- Leveling System
- Renown Rewards
- Class Bonuses
- Persistent Profiles

### 📜 Guild Activities

- Daily Rewards
- Quest Boards
- Guild Challenges

### 🔮 Rune Quizzes

- Interactive Rune Puzzles
- Hints & First Letter Reveals
- Rune Leaderboards
- Loot Rewards

### 🎒 Inventory & Economy

- Inventory System
- Item Usage
- Buying & Selling
- Guild Shop

### 🏰 Adventure

- Dungeon Runs
- Loot Drops
- Rare Rewards

### 🛡️ Collections

- Titles
- Achievements
- Leaderboards
